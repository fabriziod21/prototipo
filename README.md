# BioMarket

Prototipo clickeable de un marketplace vecinal para productores locales y vecinos compradores.

**Demo en vivo:** abre `BioMarket.html`

## Flujo principal
Vitrina → Productor → Chat → Carrito → Pago (Yape / Plin / Efectivo) → Confirmación

Incluye también: mapa de vecinos cercanos, lista de chats, y wizard de publicación express (3 pasos, <10 seg).

## Cómo verlo
1. Clona el repo
2. Abre `BioMarket.html` en un navegador moderno

O visita la versión desplegada vía GitHub Pages (Settings → Pages → Deploy from branch `main`).

## Stack
- HTML + React 18 (via CDN)
- Babel standalone para JSX
- Imágenes: Unsplash
- Sin build step — edita y recarga.
